<h1>Brute Force Tool - README</h1>
<p>Welcome to the Brute Force Tool created by Mujaheed and Ariel.</p>

<h2>About</h2>
<p>This tool is designed for educational purposes only. It helps perform brute-force attacks against web login pages by using a wordlist to attempt different password combinations.</p>

<h2>Features</h2>
<ul>
    <li>Easy-to-use command-line interface.</li>
    <li>Supports both Kali Linux and Termux.</li>
    <li>Simple brute-force attack for web-based login forms.</li>
    <li>Display of ASCII art for improved readability.</li>
</ul>

<h2>Installation</h2>
<p>1. Clone this repository or download the code files.<br>
2. Ensure the rockyou.txt wordlist is placed in the same directory as the script.<br>
3. Run the <code>brute_force_tool.py</code> script using Python 3:</p>
<pre><code>python3 brute_force_tool.py</code></pre>

<h2>How to Use</h2>
<p>1. Open your terminal or Termux.<br>
2. Navigate to the directory where <code>brute_force_tool.py</code> is located.<br>
3. Run the script:</p>
<pre><code>bash brute_force_tool.sh</code></pre>
<p>Follow the on-screen instructions to enter the target URL, username, and wordlist.</p>
